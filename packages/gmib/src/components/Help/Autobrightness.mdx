##### Рекомендации по установке автояркости
Укажите GPS-координаты экрана <LocationOnIcon />. Эти данные нужны для определения времени захода и восхода солнца,
которое используются для расчета яркости экрана при отсутствии датчика освещенности.

Задайте *контрольные точки* зависимости яркости от данных датчика освещенности. Обратите внимание,
что данные точки должны описывать *монотонную функцию* т.е. при возрастании освещенности, яркость не
может убывать. Также для задания показаний освещенности используйте логарифмическую шкалу, например:
100, 1000, 10000 люкс. Укажите эти значения даже если вы не используете датчик, в этом случае важны
только два значения — *минимальная и максимальная* яркость, а текущая яркость будет рассчитана на
основании GPS-координат.

Яркость вычисляется раз в минуту, на основании среднего значения датчика.
Если вы хотите управлять яркостью вручную, отключите режим автояркости.
Если включена автояркость, приложение автоматический запустится при входе пользователя в ОС (Windows/Mac).

Чтобы указанные изменения вступили в силу не забудьте <SaveButton /> изменения.
